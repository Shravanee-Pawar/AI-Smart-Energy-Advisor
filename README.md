# AI-Smart-Energy-Advisor

## Project Description
AI Smart Energy Advisor predicts electricity consumption using Machine Learning and provides personalized energy-saving recommendations. The application helps users estimate electricity usage, forecast electricity bills, optimize energy consumption, and promote sustainable living through an intelligent, user-friendly web platform.

## Technologies Used 
- FastAPI
- Python
- HTML, CSS, JavaScript
- Firebase
- Gemini API
- Scikit-learn

## Features
- User Registration & Login
- Dashboard
- Electricity Consumption Prediction
- Estimated Bill
- analytics
- AI Assistant
- Recommendations
- User Profile

## How to Run
1. Clone the Repository
2. Install dependencies:
   pip install -r requirement.txt
3. Add firebase-key.json to the backend folder
4. Create a .env file with
   GEMINI_API_KEY=YOUR_API_KEY
5. Run:
   uvicorn main:app --reload
